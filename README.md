# tic-tac-toe-python
A Python Tic-Tac-Toe game with Terminal &amp; Tkinter GUI, featuring Easy and Medium AI bots.

## Features
- 🎯 Human vs Bot
- 🤖 Easy Mode
- 🧠 Medium Mode
- 🖥️ Tkinter GUI
- 💻 Terminal Version
- Winner Detection
- Restart Option

## Technologies
- Python
- Tkinter
